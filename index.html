Multi-Modal Toxic Span Detection for Urdu Language

Multi-modal toxic span detection refers to identifying and detecting toxic chunks from the video, especially for the Urdu language. The proposed ... approach is firstly I break the video into frames and audio signals using Open-CV and then audio signals into text using Whisper-V2 medium. Then extract the features individually. For frames, I used VGG-19, for audio signals I used open-smile, and for text I used XLM-RoBERTa. After extracting the features, I combine all the features in a single-dimensional vector using late fusion.

Urdu Voice Command Recognition System

An Urdu voice command recognition system is a technology designed to recognize and interpret spoken commands in the Urdu language. ... This technology can be used in various applications, such as voice assistants, smart home devices, and voice-controlled software interfaces. The proposed approach is converting all audios to wav format and then extract the features using MFCC and Mel Spectograms. And after that pass to the neural network for training. The total number of classes are forty-four and this is a multiclass classification problem

Wav2Vec2 Transformer XLSR:Urdu ASR Speech Recognition in Urdu

Similar to Wav2Vec2, XLSR-Wav2Vec2 learns powerful speech representations from hundreds of thousands of hours of speech in more than 50 languages of unlabeled ... speech. Similar, to BERT's masked language modeling, the model learns contextualized speech representations by randomly masking feature vectors before passing them to a transformer network.
